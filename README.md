# Sthitha Sthapatya - স্থিত স্থাপত্য

Premium Architecture, Interior Design & Construction in Bangladesh

## Project Overview

This is a static HTML website for Sthitha Sthapatya, a premier Bangladeshi architecture, luxury interior design, and construction management firm serving homeowners and NRBs worldwide.

## Features

- **Responsive Design**: Mobile-first approach with optimized layouts
- **Modern UI**: Clean, professional design with smooth animations
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Performance**: Optimized images and lazy loading
- **Accessibility**: WCAG compliant with proper ARIA labels
- **Interactive Features**: 3D visualization demos, portfolio filtering, blog pagination
- **AI Chatbot**: Rule-based chat assistant with WhatsApp integration

## Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: CSS Grid, Flexbox, CSS Variables
- **Fonts**: Google Fonts (Playfair Display, Inter)
- **Images**: Unsplash API with WebP format
- **Deployment**: Vercel (static hosting)

## Deployment

This project is configured for deployment on Vercel:

1. **Automatic Deployment**: Connect your Git repository to Vercel
2. **Manual Deployment**: Use the Vercel CLI or drag-and-drop the project folder
3. **Configuration**: The `vercel.json` file handles routing and optimization

### Environment Setup

```bash
# Clone the repository
git clone <repository-url>
cd sthitha-sthapatya

# Start local development server
npm run dev
# or
python -m http.server 3000
```

## Performance Optimizations

- **Image Optimization**: WebP format with lazy loading
- **CSS Optimization**: Critical CSS inlined, non-critical CSS loaded asynchronously
- **JavaScript Optimization**: Deferred loading, minified code
